# Results Layout

- `calibration/`: Remy C++ vs Python-simulator calibration artifacts.
- `cpp_sim/`: dense C++ evaluations for Remy trees, PPO brains, and the exported AlphaCC policy.
- `python_sim/`: AlphaCC evolution outputs from the Python Remy-compatible simulator.

The raw cross-method comparison CSVs used for replots live in:

- `cpp_sim/dense_cpp/remy-1x.csv`
- `cpp_sim/dense_cpp/remy-10x.csv`
- `cpp_sim/dense_cpp/remy-20x.csv`
- `cpp_sim/dense_cpp/ppo-brain.*.csv`
- `cpp_sim/dense_cpp/alphacc_evolved.csv`
